LangGraph オーケストレータ エージェント 設計書

ローカル Ollama (gpt-oss) 上で動く、オーケストレータ + プランナー / エグゼキューター(並列) / エヴァリュエーター 構成の AI エージェント。各エージェントの稼働状況と思考をチャット画面でリアルタイムに可視化する。

このシステムは、LangGraph の親グラフ(オーケストレータ)が recall_memory → planner → executor(並列) → evaluator の流れを統括し、評価結果に応じて再計画ループを回す構成です。Planner が出力したタスクは Send API で複数 Executor に並列ファンアウトされ、結果はリデューサで安全にマージされます。各ノードが発行するイベントは FastAPI で SSE に変換され、React フロントの稼働パネルに「今どのエージェントが何を考えているか」をリアルタイム表示します。長期記憶は PostgreSQL + pgvector に永続化されます。
